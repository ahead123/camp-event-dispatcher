Event Dispatcher JavaScript Exercise

An global event dispatcher has already been created

And 2 handlers have been registered that will either change the background color or change the font color of this page.

You can use the console to trigger an event in order to run the registered handlers

  1. Open the JavaScript console and enter the following commands:
  2. > dispatcher.trigger("background", "blue") // To change the background color
  3. > dispatcher.trigger("font", "yellow") // To change the font color

Once you implementation for the EventDispatcher is complete, triggering those events from the console will actually change the color of this page
